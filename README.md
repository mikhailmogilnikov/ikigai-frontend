# IKIGAI Education Platform

### Prerequisites

- Bun 1.2+
- Chocolatey (Windows only)

### Installing Dependencies

```bash
make i
```

### Running in Development Mode

```bash
make dev
```

### Running a Production Build

```bash
make build
make preview
```

### Code Linting

```bash
make lint
```
